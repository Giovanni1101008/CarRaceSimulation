# Simulazione Gara Automobili in Java

## Descrizione
Progetto Java che modella automobili, motociclette e un distributore di carburante.
Il sistema simula una gara su pista tramite l'uso dei Thread.

Ogni veicolo:
- consuma carburante
- effettua pit stop
- percorre 35 giri su una pista di 1.2 km
